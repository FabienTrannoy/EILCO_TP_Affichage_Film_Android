# EILCO_TP_Affichage_Film_Android

Etudiant : Fabien Trannoy

L'application permet d'afficher les films populaires et à venir. Il permet aussi de rechercher un film par son nom, d'afficher les détails d'un film quand on le sélectionne.
Il est possible d'ajouter des films en favoris mais il ne sont pas conservés si on relance l'application.
Il est aussi possible de changer la langue de l'application.
